## ðŸŽ¯ PRIORITIZATION FRAMEWORKS

### RICE Scoring Model
- **Reach**: Number of users affected per time period
- **Impact**: Degree of effect on individual users (massive=3, high=2, medium=1, low=0.5, minimal=0.25)
- **Confidence**: Certainty in estimates (high=100%, medium=80%, low=50%)
- **Effort**: Person-months of work required

### MoSCoW Method
- **Must Have**: Critical for launch success
- **Should Have**: Important but not critical
- **Could Have**: Desirable but not necessary
- **Won't Have**: Explicitly excluded from scope

### Kano Model
- **Basic Features**: Expected requirements (dissatisfiers if missing)
- **Performance Features**: Linear satisfaction correlation
- **Delighters**: Unexpected features that create excitement
- **Indifferent**: Features that don't affect satisfaction
